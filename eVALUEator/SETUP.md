# Setup Guide

Complete guide to get the Opportunity Validator running on your machine.

## Prerequisites

- Python 3.9 or higher
- pip (Python package manager)
- Anthropic API key ([get one here](https://console.anthropic.com/))
- Git (optional but recommended)

## Installation Steps

### 1. Get the Code

**Option A: Copy from outputs** (if you have the files)
```bash
# Copy the opportunity-validator folder to your desired location
cp -r /path/to/opportunity-validator ~/projects/
cd ~/projects/opportunity-validator
```

**Option B: Clone from GitHub** (if you pushed it)
```bash
git clone https://github.com/YOUR_USERNAME/opportunity-validator.git
cd opportunity-validator
```

### 2. Create Virtual Environment (Recommended)

```bash
# Create virtual environment
python3 -m venv venv

# Activate it
# On Mac/Linux:
source venv/bin/activate
# On Windows:
venv\Scripts\activate
```

You should see `(venv)` in your terminal prompt.

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

This will install:
- `deepagents` - Agent orchestration framework
- `langchain` & `langchain-anthropic` - LLM integration
- `langgraph` - State management
- `python-dotenv` - Environment variables
- `pydantic` - Data validation

### 4. Configure API Key

```bash
# Copy example env file
cp .env.example .env

# Edit .env file and add your API key
# On Mac/Linux:
nano .env
# On Windows:
notepad .env
```

Add your key:
```
ANTHROPIC_API_KEY=sk-ant-your-key-here
```

Save and exit.

### 5. Verify Installation

```bash
# Run tests
python -m pytest tests/ -v
```

If tests pass, you're ready!

### Validate Your Own Opportunities

1. **Create a Python script:**

```python
# my_opportunities.py
from src.validator import OpportunityValidator

validator = OpportunityValidator()

my_ideas = [
    {
        "name": "Your Opportunity Name",
        "description": "What you're building",
        "icp": "Who it's for",
        "problem": "What problem it solves"
    },
    # Add more...
]

results = validator.validate_opportunities(my_ideas)
comparison = validator.compare_opportunities(results)
best = validator.recommend_next(results)

print(f"\nRecommendation: {best.opportunity.name}")
print(f"Score: {best.score.total_score}/120")
```

2. **Run it:**
```bash
python my_opportunities.py
```

3. **Review results:**
```bash
# Results are saved to:
ls opportunities/

# Each opportunity gets its own folder:
cat opportunities/Your_Opportunity_Name/validation_result.json
```

## Common Issues

### "ModuleNotFoundError: No module named 'deepagents'"

**Fix:**
```bash
# Make sure you're in virtual environment
source venv/bin/activate  # Mac/Linux
venv\Scripts\activate     # Windows

# Reinstall dependencies
pip install -r requirements.txt
```

### "ANTHROPIC_API_KEY not found"

**Fix:**
```bash
# Check .env file exists
ls -la .env

# If missing, create it:
cp .env.example .env
nano .env  # Add your key
```

### "Permission denied" when running scripts

**Fix:**
```bash
# Make scripts executable
chmod +x examples/quick_start.py

# Or run with python explicitly
python examples/quick_start.py
```

### API Rate Limits

If validating many opportunities:
```python
# Validate sequentially instead of parallel
results = validator.validate_opportunities(ideas, parallel=False)
```

This is slower but avoids rate limits.

## Cost Management

### Typical Costs
- Single opportunity: ~$0.50
- 5 opportunities (parallel): ~$2.50
- 10 opportunities: ~$5.00

### Reducing Costs

1. **Use sample data for testing:**
```python
# Free - doesn't call API
from src.models.opportunity import Opportunity
opp = Opportunity(name="Test", ...)
```

2. **Validate sequentially:**
```python
# Slower but cheaper (no parallel overhead)
results = validator.validate_opportunities(ideas, parallel=False)
```

3. **Set budget limits:**
```python
# Coming soon: max_tokens parameter
validator = OpportunityValidator(max_tokens=50000)
```
